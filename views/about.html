What Is A Colour Harmony?

If you look at colours as a geometric shape (a wheel) instead of as a spectrum, you are able to form relations between two or more colours on the colour wheel.

* VISUAL *

How Do We Find A Colour Harmony?

Geometric

One way of translating these relations into code is to place a pattern on the colour wheel with pointers on the vertices to detect what region of the colour wheel it is located at, and then the original colour would change position, the shape would contort and revolve around a central axis. RGB to RYB

This algorithm was instrumental to the development of all other colour harmony algorithms. This is because if kept in RGB, as you adjust the G value, you influence the amount of Blue that exists in a colour code (because green is made of yellow and blue). To split up and isolate these values makes the algorithms easier to develope and to understand.

1. Determine the minimum value of RGB and give it value ω

2. Subtract ω from Red, Green and Blue values

R = R - ω

G = G - ω

B = B - ω

3. Determine maximum value of RGB and give it value θ

4. Determine minimum value Red and Green and give it value γ

5. Subtract γ from Red and Green values

R = R - γ

G = G - γ

6. If Blue and Green are greater than 0, then divide B and G values by 2

7. Add G value to γ and B values

γ = γ + G

B = B + G

8. Determine the minimum value of RGB and give it value φ

9. If φ is greater than 0, then divide θ by φ and give it value Δ. Then multiply R, γ and B values by Δ

Δ = θ ÷ φ

R = R × Δ

γ = γ × Δ

B = B × Δ

10. Add ω value to R, γ and B values

11. Finished!

RYB to RGB

This algorithm is also very important because we use RYB values in algorithm calculations, but we display the conventional RGB values

1. Determine the minimum value of RYB and give it value ω

2. Subtract ω from Red, Yellow and Blue values

R = R - ω

Y = Y - ω

B = B - ω

3. Determine maximum value of RYB and give it value φ

4. Determine minimum value Yellow and Blue and give it value G;

5. Subtract G from Yellow and Blue values

Y = Y - G

B = B - G

6. If Blue and Green are greater than 0, then multiply B and G values by 2

7. Add Y value to G and R values

G = G + Y

R = R + Y

8. Determine the maximum value of RGB and give it value θ

9. If θ is greater than 0, then divide φ by θ and give it value Δ. Then multiply R, G and B values by Δ

Δ = θ ÷ φ

R = R × Δ

G = G × Δ

B = B × Δ

10. Add ω value to R, G and B values

11. Finished!
